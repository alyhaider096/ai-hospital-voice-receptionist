# Frontend Admin Dashboard

Implemented Next.js admin dashboard for official hospital appointment and call
records. The UI is designed as a professional staff console, with a soft
admin-dashboard visual direction inspired by modern Soft UI systems without
copying paid template code.

Primary views:

```txt
Dashboard overview
Appointments and status updates
Doctors
Doctor schedules
Call logs
Vapi setup readiness
Assistant guardrails
```

Design direction: calm operational dashboard, dense enough for daily staff use,
not a marketing landing page.

## Local Run

```bash
npm install
npm run dev
```

Set the backend URL with:

```txt
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8001
```
